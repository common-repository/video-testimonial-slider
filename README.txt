== Installation ==

1. Unzip the downloaded `video-testimonial-slider` zip file
2. Upload the 'video-testimonial-slider' folder and its contents into the `wp-content/plugins/` directory of your WordPress installation
3. Activate `Video Testimonial slider` from Plugins page
